# Hostel Complaint Management System

A web application for students to submit complaints about hostel facilities and for wardens to manage and resolve them efficiently.

---

## **Technology Stack**

- **Frontend:** HTML, CSS, TailwindCSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  

---

## **Features**

### Student
- Register and login  
- Submit new complaints  
- View their complaints and status  

### Warden / Admin
- Login to admin dashboard  
- View all student complaints  
- Update complaint status (Pending, In Progress, Resolved)  

---

## **Instructions to run the project**
1. Extract folder
2. Run `npm install` to install dependencies
3. Run `node server.js` or `nodemon server.js`
4. Open frontend / dashboard in browser

### Configuration
Rename `config.example.php` to `config.php` and update database credentials before running the project.
